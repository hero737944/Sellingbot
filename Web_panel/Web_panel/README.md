# TgStoreAPI - Telegram Selling Bot

A complete Telegram bot for selling digital products with UPI payment, referral system, and admin panel.

## Features

### User Features
- ✅ Welcome & Rules with compulsory channel/group join
- ✅ Mini Web App for Terms Acceptance
- ✅ Balance System (₹ + $)
- ✅ UPI Deposit with QR & Manual Approval
- ✅ Auto Product Delivery
- ✅ Referral System with ON/OFF
- ✅ Profile with purchase history
- ✅ Ban/Restrict System

### Admin Features
- ✅ Password Protected Admin Bot
- ✅ Pending Deposits Management
- ✅ Product CRUD Operations
- ✅ Payment Methods ON/OFF
- ✅ Rate Change (USDT to INR)
- ✅ Referral Settings
- ✅ Broadcast Messages
- ✅ Join Links Change
- ✅ Password Change
- ✅ Statistics Dashboard
- ✅ QR Code Change with 15 min expiry
- ✅ 5 QR Storage
